# Chess-Analyzer
This project is the incorporation of Stockfish and a UI to make a chess analyzer that will be free and effective.

It will have the capability to move the pieces on the board, find the best moves, and teach the user. 

# Current Progress: 
It shows the moves on the board with the proper labels. And it creates a String depending on the past moves to then pass onto stockfish for when i edit that
